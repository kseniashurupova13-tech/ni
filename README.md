# Random Quote Generator
**Автор:** [Шурупова Ксения]

## Описание проекта
Приложение на Python с графическим интерфейсом (Tkinter), которое генерирует случайные вдохновляющие цитаты. 

## Функционал
- Генерация случайной цитаты из встроенного списка.
- Отображение истории всех сгенерированных цитат.
- Фильтрация истории по автору или теме.
- Автоматическое сохранение истории в файл `history.json`.
- Загрузка истории при запуске приложения.

## Инструкция по запуску
1. Убедитесь, что у вас установлен Python 3.
2. Скачайте файл `main.py`.
3. Запустите его командой: `python main.py`

## Примеры использования (Тесты)
1. **Генерация:** Нажмите кнопку "Сгенерировать" — в поле сверху появится текст, а в таблице снизу добавится запись.
2. **Фильтрация:** Введите в поле "Автор" имя "Эйнштейн" и нажмите "Применить" — в списке останутся только его цитаты.
3. **Сохранение:** Закройте программу и откройте снова — история в таблице сохранится.
